# SEO og AEO

Plattformen skal gjøre hver kundeside lett å forstå for både søkemotorer og
AI-svarmotorer.

## Standard per side

- Tittel og beskrivelse på norsk, engelsk og tysk.
- Canonical URL.
- Hreflang/alternate-struktur.
- Open Graph for deling.
- Sitemap og robots.
- JSON-LD schema.

## Schema-typer i grunnmuren

- `LocalBusiness`
- `TouristTrip`
- `LodgingBusiness`
- `Article`
- `FAQPage`

## AEO-oversikt i admin

Første versjon viser en demo-score. Produksjonsversjonen bør beregne score fra:

- Antall relevante FAQ-er.
- Om artikler svarer på konkrete reisespørsmål.
- Om overnatting og aktiviteter har strukturert data.
- Om lokasjon, reisevei og praktisk informasjon er tydelig.
- Search Console-data for spørsmål og søketermer.

## Artikkelmaler

Anbefalte faste maler:

- 48 timer på stedet.
- Beste sesong å reise.
- Hva kan du gjøre i området?
- Familieguide.
- Fiske-/aktivitetguide.
- Hvordan komme hit?
